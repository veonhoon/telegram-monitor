PART 3: UNDERSTANDING THE BOT SERVER URL
================================================================================

Q: What is the Bot Server URL?
A: It's the address where your main bot server is running.

Format: http://YOUR_SERVER_IP:PORT

How to find it:
---------------
1. On your bot server machine, open Command Prompt

2. Type: ipconfig

3. Look for "IPv4 Address" - Example: 192.168.1.100

4. The port is 3000 (default) unless you changed it

5. Your URL is: http://192.168.1.100:3001

Example Setup:
--------------
Bot Server IP: 192.168.1.100
Port: 3001
Bot Server URL: http://192.168.1.100:3001

All your agents on other servers will use this URL to connect!

================================================================================
TROUBLESHOOTING
================================================================================

Problem: Agent can't connect to bot server
Solution:
  - Make sure bot server is running
  - Check firewall on bot server (allow port 3001)
  - Verify bot server URL is correct
  - Try: ping 192.168.1.100 from agent server

Problem: No ADB devices showing
Solution:
  - Run: adb devices (manually)
  - Make sure USB debugging is enabled
  - For network ADB: adb connect IP:5555

Problem: Bot not responding on Telegram
Solution:
  - Check bot token is correct
  - Make sure bot server is running
  - Check internet connection
  - Look at console for errors
